# MANTIS Documentation

Welcome to the **MANTIS** (Multi-Agency National Traffic Infringement System) documentation.

MANTIS is a unified, GIS-enabled enforcement platform built for the Fiji Islands, enabling LTA, Municipal Councils, and Fiji Police to manage traffic infringements through both a web application and mobile app.
